Fox Valley Special Recreation Association
Incident Management System

The FVSRA IMS is a web application created to replace the current paper report system for the local business, Fox Valley Special Recreation Association. This project was a part of my Software Engineering course at Aurora University. The goal of the project was to create a prototype using HTML, CSS, JavaScript, Node.js, Express, SQL, and REST architecture. It was developed in an agile sprint environment with a team of four people. The system is meant to allow employees and admins to create forms, have them update into a database, and when requested have the information output to an Excel file. Due to the nature of the course, the prototype was not meant to be fully functional. Currently, the system is semi-functional. For further information, please scroll down to the about section and select the documentation file. Overall I put in the most hours of the team, and I am proud of the progress we made. In the documentation file you will see our sprints, in which our hours committed to the project were logged. These hours only included the log of work towards the prototype itself and not towards documentation. I was responsible for the majority of the frontend, assisted with the backend and database, as well as I was the main organizer/leader of the project including duties such as organizing sprint meetings and compiling the documentation file
